Introduction

The main purpose of this practical is to give us a basic intro to direction dependent (DD) calibration as implemented in MeqTrees. If you recall from previous lectures and MeqTrees practical, that 2GC methods assume that all propagation effects are direction independent (DI) and any present DD effect is the same across all the receivers (trivial DDEs). This approximation is valid for narrower field of views but has already started showing its limitations even for smaller arrays such as the KAT7. The DD calibration technique to be applied is called the differential gains (DGs). This approach solves for G-Jones (DI) and DD effects simultaneously. It does this by estimating and correcting for receiver gain drifts (i.e GJones) in short time/frequency scales and the DGs in long time/frequency scales.

First, before we get ahead of ourselves, let us check it we have all the necessary files to perform simulations and calibrations. Log in to elwood and on your home make a new directort and give it a reason name e.g MeqTrees3GC, and then copy the files below from ... into this new directory.
  • The sky model ( summs_field.lsm.html )
  • The simms configuration config.json
  • The antenna configuration table, KAT7_ANTENNAS

When you are done copying, generate a new empty MS file: simms -jc config.json

Now open MeqTrees : meqbrowser

Press start and a "TDL Options &" window should open. Go to "TDL Options" and "Load TDL script" and naviate to Catter folder -- Siamese -- then load the turbo_sim.py .
